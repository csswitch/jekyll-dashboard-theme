## About This Site

This site is built with **Dashboard Theme** — a Jekyll theme that gives your GitHub Pages site the aesthetic of an OS desktop application.

### Features

- OS-style window chrome cards
- Sidebar navigation with active state tracking
- Stat widgets on the home page
- Reading progress bar
- Tag system with cloud + grouped list views
- Mobile sidebar toggle
- Copy-to-clipboard code blocks
- JetBrains Mono + Inter UI fonts
- GitHub dark palette

### Built With

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting
- Dashboard Theme by [csswitch](https://github.com/csswitch)
